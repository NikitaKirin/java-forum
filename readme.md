Web-приложение: Форум
=====================

Основные сущности:
------------------

-   User

    -   id

    -   name

    -   email

    -   password

-   Post

    -   id

    -   title

    -   description

    -   date

    -   user_id

-   Comment

    -   id

    -   text

    -   date

    -   user_id

    -   post_id

Логика работы приложения:
-------------------------

Пользователь заходит/регистрируется и может создавать/редактировать/удалять
собственные посты, а также комментировать их.
